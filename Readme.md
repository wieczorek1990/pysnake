snake
#####

Snake game inspired by Nokia software.

Requirements:

```
sudo apt-get install libncurses5-dev libncursesw5-dev python-dev ruby-dev python3-pip
sudo pip3 install pygame
sudo gem install curses
```

```
make
sudo make install
snake
sudo make uninstall
```

or

```
python3 snake.py
```

or

```
ruby snake.rb
```
